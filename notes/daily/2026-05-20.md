## Core Takeaways

- [ ] **Context is information governance, not text concatenation.** The model can only act on what's in its context window. If system rules, user intent, chain data, and untrusted web pages all sit at the same level, you've built a confused agent.
- [ ] **RAG is an evidence chain, not just "add a vector DB."** Chunking → vectorizing → retrieval → rerank → citation. Break one link, and hallucination just moved from inside the model to inside your retrieval system.
- [ ] **Memory ≠ permanent authorization.** What the user allowed yesterday does not hold today. All memory involving identity, assets, or external side effects must be re-bound to the current session and authorization.
